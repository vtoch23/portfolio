About me

I have been a graphic designer for the past 10 years with strong logical thinking and problem solving. My goal is to start a career in programming and to achieve this I began studying the Python language. I completed the introduction and advanced Python courses from Helsinki University, as well as Python Institute's prerequisite certification courses, followed by an Associate Certificate in Python programming.

During the courses I learned how to manipulate text files, create games and write program code using various Python modules and libraries.

I decided to explore Flask, CSS, Jinja and HTML in order to start creating web applications, as well as the use of SQL for connecting Python functionality with databases. On this website you can view some of the course exercises and other ideas which I have presented as small web projects.
