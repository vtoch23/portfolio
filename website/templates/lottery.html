Lottery

The program takes seven numbers via a form input and stores them as variables. The random module is used to produce 7 random numbers from 1 to 40 for the lottery draw, which is sent to a function along with the list of the user's numbers. The function checks each number against the drawn numbers and returns the number of hits and the matched numbers. They are sent back to the HTML as variables using Jinja syntax.

The results will appear at the bottom of the form
