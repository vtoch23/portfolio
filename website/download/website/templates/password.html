Password Generator

The program generates a random password of given number of characters. The users inputs a number via a form, which is converted to a variable and sent to the function. The function takes the number of letters as an attribute and by using the Random module and String methods, returns a password consisting of lower case letters, digits and special characters from a given list, up to the given amount of characters.
