Spellchecker

The program takes a string from an input via a form, splits it into a list of words and checks each word against a .txt file which contains a large number of common words. If the word isn't there it marks as an error and then uses difflib to provide suggestions. The misspelled word and close matches are stored in a dictionary and a for loop is used in the HTML with Jinja sytnax to loop through the values of the dictionary.

The results will appear at the bottom of the form

Please type in words and include intentional spelling errors
